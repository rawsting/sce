A Front-End Web Developer

My name is Jacob Bruce and I'm from Rome, Georgia. I'm currently enrolled at Rome High School and graduate in May 2021. I hope to become a Front End web developer and learn some Back End as well, and hopefully become a Full Stack developer eventually. I enjoy building websites from the ground up and creating beautiful webpages. I've always wanted to create an online platform or large-scale website. Outside of programming I enjoy playing soccer and currently play club soccer at SSA North Georgia and for my high school Varsity team. After graduating high school I'd like to get a entry-level job and hopefully go to college and earn a degree in Web Design.

What I do

HTML5

I have been learning and practicing HTML since late 2015. I started out on a snow day from school because I wanted to make my own Minecraft website. It started out very rough, like getting font-size mixed up with text-size, but ever since then I have been ethused in making webpages using HTML. I am very versed in making static pages and am beginning to dive into making dynamic web pages. Later this year in 2020 I will be getting certified by Microsoft with their MTA HTML5 test.

CSS3

I have been learning CSS since I started learning HTML to style it. I started out using inline CSS and gradually moved to the style tag and then to seperate stylesheets. I have grown my knowledge in CSS greatly, from forgetting how to change the font-size to an almost full understanding of Grid and Flexbox. As mentioned, I know how to use Grid and Flexbox to create beautiful layouts, as well as how to style elements and make the entire page look amazingly beautiful. I am also delving into preprocessors like SASS.

JavaScript

I have always been interested in JavaScript and have begun learning it recently. I enjoy making my pages interactive and manipulation data. I have begun learning JS5 and have delved into libraries such as JQuery which make learning and programming JS easier and quicker. I am also interested in backend libraries for JS, Node.JS specifically. I plan to become highly versed in JavaScript and learn Node.JS to delve into the backend of web development.
